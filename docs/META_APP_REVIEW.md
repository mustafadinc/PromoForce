# Meta App Review — Instagram Content Publishing

PromoForce uses **Facebook Login** → Page → **Instagram Business Account** for `instagram_content_publish`.

## Permissions

- `instagram_basic`
- `instagram_content_publish`
- `pages_show_list`
- `pages_read_engagement`

## Dev mode

Until App Review is approved, only **25 test users** can receive published posts. Add testers in Meta Developer Console → Roles.

## Production checklist

1. Privacy policy URL on marketing site
2. Screen recording: OAuth connect → generate post → publish to IG
3. Explain data usage: tokens stored encrypted, used only to publish on user request
4. Submit for review from App Dashboard → App Review

Typical timeline: **2–6 weeks**.
